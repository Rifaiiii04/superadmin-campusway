Konfigurasi Database untuk TKA SuperAdmin:

1. Buat database MySQL dengan nama: arahpotensi
2. Buat user MySQL dengan permission yang sesuai
3. Copy file .env.example ke .env dan sesuaikan konfigurasi:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=arahpotensi
DB_USERNAME=root
DB_PASSWORD=

4. Jalankan migration:
   php artisan migrate

5. Generate app key:
   php artisan key:generate

6. Jalankan server:
   php artisan serve
